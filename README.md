# TailwindCSS Coffee Style

Live Demo: :point_right: [Click me](https://coffee-tailwind-minatisleeping.web.app/)


## Requirements

* **node >= v16.14.0**
* **npm >= v6.14.7**
* **yarn >= v1.9.1**

Clone project & runt test

```
    $ git clone https://git_url_clone <project_dir>
    $ cd <project_dir>
    $ yarn build:css
```
